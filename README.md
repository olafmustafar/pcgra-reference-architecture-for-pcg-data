# pcgra-reference-architecture-for-pcg-data
Repository for storing supplementary data of the paper "PCGRA: A Reference Architecture for Procedural Content Generation in Digital Games"

Files are presented as follows:

| file                                                   | description                                        | language |
|--|--|--|
| Apresentacao PCG-RA Video.mkv                          | file used in the PCGRA presentation during PROSARA | pt |
| Apresentação PCG-RA Slides.pdf                         | file used in the PCGRA presentation during PROSARA | pt |
| Apresentação PROSA-RA.docx                             | file used in the PCGRA presentation during PROSARA | pt |
| PROSARA checklist translated.pdf                       | results from the checklist performend in PROSARA   | pt |
| PROSARA checklist.pdf                                  | checklist translated to english                    | en |
| Apresentação PCG-RA V1.docx                            | old PCGRA design document                          | pt |
| Apresentação PCG-RA V1.pdf                             | old PCGRA design document                          | pt |
| Presentation PROSA-RA.docx                             | new PCGRA design document, portuguese and english  | pt/en |
| PCGRA benchmark.xlsx                                   | benchmarks results                                 | en |
| survey/graph**                                         | graphs derived from the survey results             | en |
| survey/Formulario A Jornada de Hercules.csv            | raw results from the survay                        | pt |
| survey/Formulario A Jornada de Hercules_translated.csv | raw results translated to english                  | en |
